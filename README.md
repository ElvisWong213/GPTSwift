# GPTSwift

# About this project
Use ChatGPT on Mac and iOS devices

# Framework Used
- SwiftUI
- SwiftData
- Keychain
- ChatGPT API

# Requirements
- iOS 17+
- MacOS 14+
- Swift 5+
